Privacy Policy

Last updated: 2025-09-17

Resume Checker analyzes your CV and a job description to provide a structured analysis and a match score.

Your inputs (CV and job description) are transmitted over HTTPS to https://cv.kroete.io solely to perform the analysis and return results. We do not inject code into websites and we do not collect browsing data.

Data We Process

  • CV PDF: Provided by you and sent over HTTPS to https://cv.kroete.io solely for analysis. Processed transiently and deleted immediately after analysis. We do not retain CVs on the server.
  • Job description text: Provided by you, sent over HTTPS for analysis and scoring; may include information you include in the text.
  • API key: Your OpenAI API key is stored locally in the extension via chrome.storage.local and sent to the backend in a request header to complete the analysis. It is not stored server-side.
  • Session cache: The extension may temporarily cache the selected CV file in chrome.storage.session during the current browser session to improve UX; this is cleared when the session ends.

How We Use Data

  • To generate structured CV and job analyses and compute a match score.
  • No advertising or sale of data. We do not share data with third parties except necessary processors strictly to provide the service (e.g., your selected LLM provider when invoked via your API key).

Retention

  • CV PDFs are deleted immediately after analysis.
  • We do not store results or personal data beyond the request lifecycle. Minimal operational metadata (e.g., timestamps or status codes) may be retained for reliability and abuse prevention; no content is stored.

Security

  • All production traffic is encrypted in transit (HTTPS).
  • The extension’s Content Security Policy allows scripts from 'self' only; no remote scripts or Wasm are executed.
  • We recommend not including sensitive personal identifiers in CVs beyond what is necessary.

User Controls

  • You can view and remove your API key from the extension’s Options page at any time.
  • You can clear session data by closing the browser session or uninstalling the extension.
