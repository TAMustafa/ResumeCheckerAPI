Privacy Policy

Last updated: 2025-09-08

Resume Checker analyzes your CV and a job description to provide a structured analysis and a match score.

Data We Process

  • CV PDF: Uploaded solely for analysis. The file is processed transiently and deleted immediately after analysis. We do not retain CVs on the server.
  • Job description text: Sent for analysis and scoring.
  • API key: Your OpenAI API key is stored locally in the browser extension and sent to the backend as a request header to perform analysis. It is not stored server-side.

How We Use Data

  • To generate structured CV and job analyses and compute a match score.
  • To return results to you; we do not sell data or share it with third parties.

Retention

  • CV PDFs are deleted immediately after analysis.
  • We do not store results or personal data beyond the request lifecycle. In-memory caches may hold non-sensitive derived data briefly to reduce cost/latency.

Security

  • All production traffic is served over HTTPS.
  • We recommend not including sensitive personal identifiers in CVs beyond what is necessary.
